# Library-Management-System

Simplify book and borrower management for Bosta task. This repository hosts the codebase for an efficient system built with Nodejs, Nestjs, and Express.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev
```
