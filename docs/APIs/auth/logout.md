# Routes

## POST /api/v1/auth/logout

Logout user.

### Response

If the request is success, the server will respond with a status code of 200:

```json
{
  "status": 200,
  "message": "Logout Successfully",
  "data": null,
  "total": 0,
  "meta": {}
}
```
